Comparison: Latitude vs. Max Temperature/Humidity/Cloudniness/Windspeed

The purpose of this page was to analyse and visualise the weather from 500+ cities across the world with varying distance from the equator. To accomplish this analysis, the writer pulled data from the OpenWeatherMap API to get the data set of 500+ random cities across the world.


After the data set was pulled, the writer utilised our MatPlotLib to plot various relationships of the weather vs. latitude. Relationships explored including: temperature, cloudiness, wind speed and humidity.
