Summary: Latitude vs. X

In this project, the purpose was the analyze and visualize the weather of 500+ cities across the world of varying distance from the equator. T accomplish this analysis, we pulled data from the OpenWeatherMap API to get our data set of 500+ random cities across the world.


After the data set was pulled, we utilized our MatPlotLib skills to plot various relationships of the weather vd. latitude. Relationships we explored were: temperature, cloudiness, wind speed and humidity. This site has the source data and visualizations created during this analysis and explanations of the relationships observed.
